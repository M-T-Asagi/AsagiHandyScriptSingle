# 自作便利スクリプト（主にUnity向け）

自分がUnity上で開発している際に作ったものをそれとなく使えるようにしておいてあるもの。

すべてMITライセンスなので、利用についてはMITライセンスに従ってください。

## ScriptsUnityInputting
https://github.com/M-T-Asagi/ScriptsUnityInputting

主に入力系の処理についてまとめてあるリポジトリ

## ScriptsUnityObjectControl
https://github.com/M-T-Asagi/ScriptsUnityObjectControl

Unity上でオブジェクトを操作したりするスクリプトがまとめてある

## ScriptsUnityUIControl
https://github.com/M-T-Asagi/ScriptsUnityUIControl

UI系の処理

## そのほか
https://github.com/M-T-Asagi/ScriptsUnityUtil

上記三項目に当てはまらないようなスクリプト

数学関数系や画面の回転を検知したりするものなど、雑多
